# 🛡️ Mini SOC Monitoring System

A Security Operations Center (SOC) simulation built with Python and Flask that detects suspicious activities like brute-force attacks and port scanning.

---

## 🔍 Features
- Log monitoring system
- Brute-force detection
- Port scan detection
- Security alert engine
- Web dashboard (Flask)
- Clean UI

---

## 🧠 Project Idea
This project simulates how real SOC systems work by analyzing logs and generating security alerts based on attack patterns.

---

## ⚙️ Tech Stack
- Python
- Flask
- HTML / CSS
- JSON storage

---

```bash
pip install flask
python app.py
