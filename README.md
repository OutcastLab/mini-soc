# 🛡️ Mini SOC Monitoring System

A simulated Security Operations Center (SOC) built with Python and Flask.

## 🔍 Features
- Real-time log monitoring
- Brute force detection
- Port scanning detection
- Security alert classification (High / Medium)
- Web-based SOC dashboard

## 🧠 Concept
This project simulates how real SOC systems monitor logs, detect suspicious behavior, and generate security alerts.

## ⚙️ Tech Stack
- Python
- Flask
- HTML/CSS
- JSON-based storage
